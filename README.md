Diabetes Prediction System (82% Accuracy)
Este proyecto combina Data Science y Web Development para predecir el riesgo de diabetes basándose en el dataset de Pima Indians.

🧠 Machine Learning Insights
Optimización: Se logró elevar la precisión del 76% al 82% mediante una limpieza selectiva de características.

Feature Selection: Se eliminaron variables con ruido o baja correlación como BloodPressure, SkinThickness y DiabetesPedigreeFunction.

Balanceo de Datos: Implementación de SMOTE para corregir el desequilibrio de clases en el dataset original.

Modelo: RandomForestClassifier con parámetros ajustados (min_samples_split=10, class_weight='balanced').

🛠️ Stack Tecnológico
Frontend: Next.js (React), Tailwind CSS.

Backend: Python (Flask/FastAPI), Scikit-Learn, Joblib.

Deployment: GitHub, Render/Vercel.
